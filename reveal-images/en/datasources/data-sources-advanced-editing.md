# Advanced Information Editing for Data Sources

*The advanced editing of a data source* allows you to modify what data sets and data fields other users can see and work with directly inside Analytics. This could be useful in scenarios when you don't want or can't modify the original data source. The advanced editing might not be available due to [limitations](#advanced-editing-limitations) related to certification or the data source type.
