# Uploading Dashboards

In Analytics, you can also work with dashboards saved on your computer/device by directly uploading them. To do that, perform the following steps:

1. Go to the <img src="images/dashboards-icon.png" alt="The dashboards list icon" width="25px" height="25px"/> *Dashboards* tab on the left.
2.  Select the three dots menu icon     <img src="/reveal-images/en/datasources/images/overflow-icon.png" alt="Overflow icon" width="25px" height="25px"/> on the left of the *+ Dashboard* blue button.
3.  Choose _Upload_.
4.  A dialog showing your local files will open. Double click/tap the dashboard you want to upload. Dashboards' file extension is **.rdash**.
    >[!NOTE]
    >**Uploading ReportPlus Dashboards** Analytics also allows you to upload and work with dashboards created in ReportPlus. ReportPlus dashboards' file extension is **.rplus**.

Your dashboard is now uploaded and ready to be edited and shared with others.
