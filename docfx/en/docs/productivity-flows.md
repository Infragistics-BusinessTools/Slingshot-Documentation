# Productivity Flows

Productivity has always been a top priority for businesses and individuals but lately is getting more attention than ever before.
Remote work creates a deeper need for all-in-one apps like Slingshot, which get rid of constant app switching. That alone can help but productivity flows are also important to foster engagement, efficiency and, above all, save time.  

## What's a Productivity Flow
It's a way of using Slingshot that takes advantage of the seamless integration between different features, helping you get more job done. This is a great way to increase your efficiency and enhance your experience with Slingshot.

## List of Scenarios

Below you can find a list of scenarios that can make a real difference for you and boost your productivity.:

| Category | Productivity flow | Example Scenario | How To Use|
|---|---|---|---|
| Task/Chat | Start a discussion from a task | While reviewing your tasks, you discover a task important for the whole team that is labeled as low priority. You quickly realize you need to discuss it with the team right away. | Go to the overflow menu of the task >  choose "Start a Discussion" > navigate to the workspace or project where you want to start a new discussion or use an existing one.  |
| Tasks/Discussions | Start a chat from a task | Imagine you are reviewing your tasks list. You are assigned a task with  another Slingshot user. You need to have a quick conversation about your progress so you prefer to contact your task collaborator in a private message. | overflow menu of the task > "Start a Private Chat" >  choose an existing  chat or start with a new user. |
| Discussions | Create a task from a discussion | Imagine you and your teammates have finally found a common ground in a heated discussion. In the final messages you all decide on an action plan. You want to directly use these messages to create tasks without leaving the discussion topic. | overflow menu > "Create a Task" > navigate to a list of tasks > task creation menu. |
| Pins | Start a private chat from a file | Imagine you are working on a spreadsheet pinned to your team board. You need a teammate's help for completing the information inside the spreadsheet. You prefer a quick update in a private chat. | overflow menu of the file > "Start a Private Chat" > Set File permissions if the file was originally added to the board by you > the file appears attached to the chat message |
| Pins | Create a task from a file | Imagine you are reviewing the content pinned to your boards. You find a file that needs your immediate attention but you haven't finished cleaning up your content.  You want to create a task without leaving the Content tab. | overflow menu of the file > "Create a Task" > Navigate to a list of tasks > task creation menu.  |
| Pins | Create a discussion/topic from file | You notice a file in your Team's content and immediately decide you need to start a discussion about its purpose. You can do this without even leaving the Content tab of your team and interrupting your productivity flow. | overflow menu of the file > "Start a Discussion" > Navigate to the desired location. |
| Analytics | Pin a dashboard to board | There are KPIs you and your team need to track on a regular basis. You have the perfect dashboard created with Slingshot Analytics. Pin it to your team's board so it's quickly accessible by everybody. | overflow menu of the dashboard > "Pin to Board" > Navigate to the desired Content board. |
| Analytics | Start a private chat from a dashboard | You want to briefly discuss with a colleague the change of a KPI tracked in a dashboard. For this purpose, you can share your dashboard in a chat message. Plus, you can quickly add a screenshot of the visualization displaying the said KPI. Do all of this without leaving the Analytics tab. | overflow menu of the dashboard > "Start a Private Chat" > Choose an existing private/group chat or start a new one. |
| Analytics | Create a task from a dashboard | Imagine you are analyzing the information on a dashboard in your Analytics tab. You are already itching to create some tasks and solve the problems you found. Well, this is great, because you don't even need to leave the Analytics tab to create the tasks you want and continue your dashboard analysis afterwards!  | overflow menu of the dashboard > "Create Task" > Navigate to the desired Tasks list > Task Creation dialog. |
| Analytics | Start a discussion/topic from a dashboard |  Imagine you are analyzing your dashboards in the Analytics tab. You need to raise awareness on the increased support cases. To attract everybody's attention, you decide to start a discussion. | overflow menu of the dashboard > choose "Start a Discussion" > Navigate to the desired location оf your discussion |
| Private Chat | Create a task from a private message (PM) | Imagine you are chatting with a teammate. You just found the solution of an issue together. Now, you want to quickly add a task to take action after you finish your conversation. You can do this without leaving the private chat.  | overflow menu of the private message > "Create a Task" > Navigate to the desired task list  |
| Attachments | Pin to Board, Create a Task or Start a Discussion from an attachment in a PM | All attachments sent in a private message can be used by all participants to start a productivity flow without even leaving the chat. You can use each attachment independently of the text in the PM. You can save it to a board, create a task or start a discussion.  | overflow menu of the attachment > "Pin to Board" / "Create a Task" / "Start a Discussion" > navigate to the desired board/tasks list/ discussion |
| Attachments | Pin to Board, Start a Private Chart or Start a Discussion from an attachment in a task | All attachments in a task can be used independently of the task to start a productivity flow. You can save an attachment to a board, create a task or start a discussion about it. | overflow menu of the attachment > "Pin to Board" / "Start a Private" / "Start a Discussion" > navigate to the desired board/ chat / discussion |
| Bookmarks | Pin to Board, Start a Private Chat, Create a Task or Start a Discussion from bookmark | You can bookmark everything in Slingshot - messages, tasks, files, attacments, etc. You can use any bookmark to start a private chat, discussion, create a task or pin it to  a board. | overflow menu of the bookmark > "Pin to Board" / "Start a Private Chat"/ "Create a Task" / "Start a Discussion" > navigate to the desired board/chat / tasks list / discussion |
