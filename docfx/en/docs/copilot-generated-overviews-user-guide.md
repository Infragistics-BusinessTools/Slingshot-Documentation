# Overviews User Guide

Overviews help you monitor your work, track progress, and access critical information at a glance. Whether managing personal tasks or leading multiple teams, Overviews provide a customizable workspace to stay organized and informed.

## What Is This?

Overviews are customizable dashboards that consolidate information from your workspaces, projects, and tasks into one unified view. They answer key questions like:

* Are we on time?
* What issues need attention?
* Who is working on what?
* Where are important documents and resources?
* What metrics matter most right now?

Overviews combine widgets, filters, and layouts to create personalized information hubs for individuals and teams.

## Why Use Overviews?

Overviews help you:

* **Make informed decisions** with real-time data visualization
* **Track overall progress** across multiple projects
* **Streamline workflows** by centralizing information
* **Collaborate effectively** with shared team views
* **Stay organized** with personalized work summaries

## Overview Types

Slingshot provides three types of Overviews:

### My Overviews

Personal dashboards visible only to you. Located in the top left corner of the screen, My Overviews help you organize your work and visualize tasks in a summarized way.

* Private and customizable
* Focus on your individual tasks and priorities
* Accessible from anywhere in Slingshot

### Workspace Overviews

Shared dashboards that provide a snapshot of workspace information. Workspace Overviews help teams and leaders monitor overall status and highlight important content for all members.

* Shared with all workspace members
* Default overview can be customized by owners
* Ideal for team-level collaboration

### Project Overviews

Project-specific dashboards that focus on single project details. Each project comes with a default overview that can be customized to fit team needs.

* Specific to individual projects
* Only visible to project members
* Customizable by users with owner permissions

> [!NOTE]
> Free version users can only edit the default overview in My Overviews, workspaces, and projects. Slingshot and Slingshot Enterprise subscriptions allow you to create and edit multiple overviews.

## Getting Started

### Access Requirements

To access Overviews:

* **My Overviews**: All users can view and edit personal overviews
* **Workspace/Project Overviews**: Must be a member of the workspace or project
* **Editing Workspace/Project Overviews**: Requires owner permissions

### Permissions Overview

| Action | My Overviews | Workspace/Project (Owner) | Workspace/Project (Member) |
|--------|--------------|---------------------------|----------------------------|
| View | ✓ | ✓ | ✓ |
| Edit | ✓ | ✓ | ✗ |
| Add Widgets | ✓ | ✓ | ✗ |
| Remove Widgets | ✓ | ✓ | ✗ |
| Create New Overview | ✓* | ✓* | ✗ |

*Requires Slingshot or Slingshot Enterprise subscription

### First-Time Setup

When you first open Slingshot:

1. Click **My Overviews** in the top left corner
2. Review the default overview created for you
3. Click the pencil icon to enter edit mode
4. Add or remove widgets to match your workflow

## Creating Your First Overview

Follow these steps to create a personalized overview:

### Step 1: Open My Overviews

Click **My Overviews** in the top left corner of Slingshot.

### Step 2: Enter Edit Mode

Click the pencil icon in the top right corner to enable editing.

### Step 3: Add Your First Widget

1. Click **+ Widget**
2. Choose from popular widgets or click **Widget Library** to see all options
3. Select a widget that matches your needs

### Step 4: Configure the Widget

1. Click the widget's overflow menu (three dots)
2. Select **Edit**
3. Configure settings on the left side of the widget
4. Click the checkmark to save changes

### Step 5: Organize Your Layout

* Drag widgets to reposition them
* Resize widgets by dragging corners or edges
* Arrange widgets for optimal viewing

### Step 6: Save Your Overview

Click the checkmark in the top right corner to save all changes.

> [!TIP]
> Start with 3-5 widgets focused on your daily priorities. You can always add more later.

## Customization Options

### Configuring Views

Task widgets support three view options:

* **Chart**: Visual representation of task data with graphs and charts
* **Split**: Combined view showing both chart and task list
* **Table**: Detailed list view with columns and rows

To change a widget view:

1. Click the zoom icon on a task widget to maximize it
2. Click the **View** dropdown
3. Select your preferred view
4. The view saves automatically for all users

> [!NOTE]
> List, Timeline, and Calendar visualizations do not support view switching.

### Applying Filters

Filter task widgets to focus on specific data:

1. Click the zoom icon to maximize a task widget
2. Click the filter icon next to **View**
3. Select filter criteria (status, assignee, date range, etc.)
4. Click **Apply** to save the filter

Filters help narrow down information to what matters most at any given time.

> [!NOTE]
> Timeline and Calendar visualizations do not support filtering.

### Layout Management

Customize your overview layout:

* **Drag and Drop**: Click and drag any widget to a new position
* **Resize**: Hover over widget edges and drag to resize
* **Grid Alignment**: Widgets snap to a grid for clean alignment
* **Responsive Design**: Layouts adjust automatically on different screen sizes

### Refresh Options

Overviews update automatically every 30 minutes. To manually refresh:

* Click the **Refresh** button in the top right corner
* Data updates immediately across all widgets

## Widgets

Widgets are the building blocks of Overviews. Slingshot provides a comprehensive widget library organized into six categories.

### Widget Library Categories

#### Task Management

Track and visualize tasks with these widgets:

* **Tasks by Status**: View task distribution across statuses
* **Open Tasks**: See all tasks not yet started
* **In Progress Tasks**: Monitor currently active tasks
* **Overdue Tasks**: Identify tasks past their due date
* **Blocked Tasks**: Find tasks waiting on dependencies
* **Tasks Completed Over Time**: Track completion trends

#### Pins & Content

Access important content quickly:

* **Pins**: Display pinned items for quick access
* **Bookmarks**: Show bookmarked content and resources

#### Dashboards & Analytics

Integrate dashboard data:

* **Pin from Dashboard**: Embed specific dashboard visualizations
* **My Favorite Dashboards**: Quick links to your favorite dashboards
* **Dashboards**: Access and manage dashboards

#### Projects & Workspaces

Monitor project status:

* **Projects Timeline**: View project schedules and milestones
* **Open Tasks by Project**: See task distribution across projects

#### Informational

Display team and task information:

* **Member Tasks Summary**: Overview of tasks assigned to team members
* **Member Tasks by Status**: Task status breakdown by team member
* **Text Widget**: Add custom text, notes, or instructions

#### Discussions & Collaboration

Stay connected with your team:

* **Unread Mentions**: Track mentions and notifications requiring attention

### Adding a Widget

1. Open an overview
2. Click the pencil icon to enter edit mode
3. Click **+ Widget**
4. Select a widget from the list or open **Widget Library**
5. Choose your desired widget
6. Click the checkmark to add it to your overview

### Editing a Widget

1. Enter edit mode by clicking the pencil icon
2. Click the overflow menu (three dots) on the widget
3. Select **Edit**
4. Modify settings on the left panel
5. Click the checkmark to save changes
6. Click the checkmark in the top right to update the overview

> [!NOTE]
> Pins, Pin from Dashboards, Bookmarks, Unread Mentions, and My Favorite Dashboards widgets cannot be edited.

### Duplicating a Widget

To create a copy of an existing widget:

1. Click the overflow menu on the widget
2. Select **Duplicate**
3. Edit the duplicated widget as needed
4. Click the checkmark to save

> [!NOTE]
> Pins, Pin from Dashboards, Bookmarks, Unread Mentions, and My Favorite Dashboards cannot be duplicated.

### Removing a Widget

1. Click the overflow menu on the widget
2. Select **Remove**
3. The widget is removed immediately

### Custom Widgets

Create custom widgets to meet specific needs:

1. Click **+ Widget** in edit mode
2. Select **Create Custom Widget**
3. Choose widget type (Task, Workspace, or Project)
4. Configure data sources and display options
5. Save your custom widget

For detailed information on custom widgets, see [Custom Widgets](custom-widgets.md).

## Sharing & Collaboration

### Workspace and Project Overviews

Workspace and project overviews are automatically shared with all members. Changes made by owners are visible to everyone with access.

### Collaboration Best Practices

* **Align on Key Metrics**: Work with team leads to identify the most important information to display
* **Keep It Focused**: Avoid cluttering overviews with too many widgets
* **Regular Reviews**: Schedule periodic reviews to ensure overviews remain relevant
* **Communicate Changes**: Let team members know when you update shared overviews
* **Respect Permissions**: Only owners should modify workspace and project overviews

### Team Coordination

Use overviews to improve team coordination:

* Display team member task summaries to show workload distribution
* Pin important project timelines for visibility
* Highlight blocked tasks that need attention
* Showcase completion trends to celebrate progress
* Share unread mentions to ensure timely responses

> [!NOTE]
> Overviews cannot be shared via link. Access is controlled by workspace and project membership.

## Troubleshooting & FAQ

### Common Issues

#### Widgets Not Loading

**Problem**: Widgets appear empty or fail to load data.

**Solution**:
1. Click the Refresh button to manually update data
2. Check your internet connection
3. Verify you have access to the underlying data source
4. Try closing and reopening the overview

#### Cannot Edit Overview

**Problem**: Edit button is not available or changes cannot be saved.

**Solution**:
* Verify you have owner permissions for workspace/project overviews
* Check your subscription level (Free accounts have limitations)
* Ensure you are not in view-only mode
* Contact your workspace owner to request permissions

#### Layout Changes Not Saving

**Problem**: Widget positions reset after saving.

**Solution**:
1. Ensure you click the checkmark to save after rearranging
2. Wait for the save confirmation before navigating away
3. Try clearing your browser cache
4. Refresh the page and try again

#### Filters Not Working

**Problem**: Applied filters do not change displayed data.

**Solution**:
* Click **Apply** after selecting filter criteria
* Verify the widget supports filtering (Timeline and Calendar do not)
* Check if any conflicting filters are applied
* Refresh the widget data

### Frequently Asked Questions

**Q: How many overviews can I create?**

A: Free accounts can edit default overviews only. Slingshot and Slingshot Enterprise subscriptions allow unlimited custom overviews.

**Q: Can I share my personal overview with others?**

A: No, My Overviews are private. To share an overview, create it at the workspace or project level.

**Q: How often do overviews update?**

A: Overviews auto-refresh every 30 minutes. Use the Refresh button for immediate updates.

**Q: Can I export overview data?**

A: Individual widgets may support data export. Check the widget's overflow menu for export options.

**Q: What happens to overviews when I leave a workspace?**

A: You lose access to workspace and project overviews when you leave. Your personal overviews remain unchanged.

**Q: Can I use the same widget multiple times?**

A: Yes, duplicate widgets or add the same widget type multiple times with different configurations.

**Q: Why can't I find a specific widget?**

A: Use the search bar in the Widget Library to locate widgets by name. Some widgets may require specific subscriptions or permissions.

**Q: How do I restore a deleted widget?**

A: Deleted widgets cannot be restored. Add the widget again from the Widget Library and reconfigure it.

**Q: Do changes to workspace overviews affect all members?**

A: Yes, all workspace members see the same workspace overview. Changes made by owners are visible to everyone.

**Q: Can I create overviews for archived projects?**

A: No, overviews are only available for active workspaces and projects.
