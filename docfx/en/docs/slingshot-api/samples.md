# Slingshot Outlook Add-in (React) 

If you want to see how the Slingshot's API can be used in our data-driven world, you can find our public sample <a href="https://github.com/Infragistics/Slingshot/tree/main/Samples" target="_blank">here</a>. This sample creates a Microsoft Outlook addin. It gives you the ability to turn any email into a Slingshot task, right from you inbox. 

Alternatively, if you want to manually try out the API, you can do that by using our [API Explorer](api-explorer.md).
