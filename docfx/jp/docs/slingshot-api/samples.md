# Slingshot Outlook アドイン (React)

データ駆動型の世界で Slingshot の API をどのように使用できるかを確認したい場合は、公開サンプルを<a href="https://github.com/Infragistics/Slingshot/tree/main/Samples" target="_blank">こちら</a>で見つけることができます。このサンプルでは、Microsoft Outlook アドインを作成します。受信トレイから直接、あらゆるメールを Slingshot タスクに変換できます。 

または、API を手動で試してみたい場合は、[API エクスプローラー](api-explorer.md)を使用して実行できます。
