# 一般的な Slingshot リソース

### 項目オブジェクト

タイプ T のオブジェクトの配列 (**タスク、ピン固定、ディスカッションなど**) を含み、複数のリソース要求 (*GetAll*) で返されます。

| プロパティ | 型|
-------------------------------------------------------------------- | ------------------ | ------------------ | ------------------ |
|items|配列|

### ドキュメント情報オブジェクト 

特定のドキュメントに関する情報が含まれています。

| プロパティ | 型|
-------------------------------------------------------------------- | ------------------ | ------------------ | ------------------ |
|id|文字列|
|name|文字列|

### メンバー情報オブジェクト

特定のメンバーに関する情報が含まれています。

| プロパティ | 型|
-------------------------------------------------------------------- | ------------------ | ------------------ | ------------------ |
|id|文字列|
|email|文字列|
|name|文字列|
|role|文字列列挙体 [(管理者、編集者、閲覧者)](https://www.slingshotapp.io/ja/help/docs/security)|

### 割り当て先情報オブジェクト

特定のタスク割り当て先などのに関する情報が含まれています。

プロパティ | 型|
-------------------------------------------------------------------- | ------------------ | ------------------ | ------------------ |
|id|文字列|
|email|文字列|
|name|文字列|

### チャット メッセージ情報オブジェクト

特定のチャット メッセージに関する情報が含まれています。 

プロパティ | 型|
-------------------------------------------------------------------- | ------------------ | ------------------ | ------------------ |
|id|文字列|
|userId|文字列|
|timestamp|double|
|text|文字列|
