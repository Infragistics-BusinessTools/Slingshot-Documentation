# データ制限

ダウンロードされたファイルのサイズ、結果セット内のセルの数 (集計後) およびピボット テーブルとグリッドのサイズ (セルの数として指定) に関して、Slingshot Web を使用する場合のサーバー側のサイズ制限があります。これらの制限の目的は、サーバーのリソース (メモリとディスク領域) が不足するのを防ぐことです。

## デフォルトの制限値:

- CSV/JSON/Excel をダウンロードする場合は 200mb
- 1000 万のセル
- 6400 万文字 (すべてのセルにすべての文字列を追加)

制限を増やしたい場合は、<a href="https://account.slingshotapp.io" target="_blank">こちら</a>でアカウントにログインし、リクエストを送信してください。私たちのチームがさらにお手伝いいたします。
